# 100daysv3

A MEVN stack.  Mongo (Mongoose ORM), Express, Vue and Node.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run server to connect to Mongo
```
npm run server
```
